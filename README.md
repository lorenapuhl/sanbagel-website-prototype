# Blackstudio Official - Website Prototype

A modern, minimalist website prototype for Blackstudio, a dance school in Mexico City. This site features an embedded Instagram feed to automatically display the studio's latest content.

## Project Overview

This prototype was designed to establish an online presence for Blackstudio while leveraging their active Instagram account for dynamic content. The site includes:

- Clean, dark-themed design matching Blackstudio's brand aesthetic
- Responsive navigation menu
- Embedded Instagram feed showing latest posts
- Contact information and address

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Montserrat)
- Fouita Instagram Widget
- GitHub Pages (hosting)

## Design Choices

- **Dark background**: Matches the studio's bold Instagram aesthetic
- **Montserrat font**: Clean, geometric sans-serif that complements the brand
- **Full-width Instagram feed**: Shows dynamic content without manual updates
- **Centered layout**: Creates focus on the studio name and social content
