# Website Prototype

A modern, minimalist website prototype. This site features an embedded Instagram feed to automatically display the business' latest content.

## Project Overview

This prototype was designed to establish an online presence for instagram-based businesses while leveraging their active Instagram account for dynamic content. The site includes:

- Clean,aesthethic and minimalist
- Responsive navigation menu
- Embedded Instagram feed showing latest posts
- Contact information and address
